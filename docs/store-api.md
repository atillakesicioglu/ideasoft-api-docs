Listeleme işlemi, sayfalama (`limit`, `page`), sıralama (`sort`), tarih aralığı (`startCreatedAt`, `endCreatedAt`),   durum (`status`), cihaz türü (`deviceType`), para birimi (`filterCurrency`), fiyat aralığı (`minPrice`, `maxPrice`) ve diğer parametrelerle özelleştirilebilir.
